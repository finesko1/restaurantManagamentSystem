# Простое приложение управления заказами

## Инициализация пакетов node
### npm i

## Инициализация пакетов laravel
```bash
  npm install
```

## Настройте .env

## Инициализируйте БД
```
    php artisan migrate
```

## Тестовые данные
```
    php artisan db:seed
```

## Запуск приложения
```
    php artisan serve
```
```
    npm run dev
```
```
    php artisan queue:work (для слушателей)
```

# Простое приложение управления заказами

## Инициализация пакетов node
```bash
  npm install
```

## Инициализация пакетов laravel
```bash
  composer i
```

## Настройте .env

## Инициализируйте БД
```bash
    php artisan migrate
```

## Тестовые данные
```bash
    php artisan db:seed
```

## Запуск приложения
```bash
    php artisan serve
```
```bash
    npm run dev
```
```bash
    php artisan queue:work
```
